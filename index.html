SC.NUMERIQ.TECHNOL.
Les élèves n'ayant pas leur score PIX indiqué dans Pronote doivent au plus vite saisir le code ci-dessous que je puisse récupérer leur profil. Code PIX : NQYKXM785 Une fois le profil enregistré ; il faut que votre score soit sur Pronote ; vous avez jusqu'au vendredi 20 février pour gagner des points. 1 point sur 20 = 10 points PIX Par conséquent pour avoir 20/20, il faut gagner 200 points PIX. Un conseil pour que votre score augmente plus vite, concentrez-vous sur les items où vous avez le plus petit niveau : cela va généralement assez vite. Je vous transmettrai un autre code lorsque j'aurai récupéré tous les profils pour valider votre progression. Tout retard dans la remontée du profil entrainera la perte de 1 point sur 20 par jour de retard. Par exemple, si vous transmettez votre profil le dimanche 22 février, vous perdrez 2 points sur 20. Bon courage !
